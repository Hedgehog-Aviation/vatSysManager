## hedgehog-aviation/vatManager-fork
### Fork of the VatSys launcher and plugin/profile manager
#### Installation Instructions (portable only):
- Download the zip archive from the [latest release](https://github.com/hedgehog-aviation/vatmanager-fork/releases/latest)
- Unblock the zip file by right-clicking on it, selecting Properties, then Unblock
- Extract the zip into a directory of your choice
- Run VatSysManager.exe and enjoy!

#### Contribute
If you have a vatSys plugin and would like it to be in the VatSys Manager, PRs are welcome - just add your plugin URL and DLL path into [plugins.json](https://github.com/Hedgehog-Aviation/vatManager-fork/blob/master/vatSysManager/Plugins.json)
